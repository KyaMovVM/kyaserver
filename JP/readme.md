Что такое словари?
Словари — это созданные пользователями наборы слов, фраз или буквосочетаний, которые можно использовать в качестве заданий для набора в игре. Полный список приведен в разделе «Словари». Чтобы начать игру по словарю, зайдите с главной страницы в раздел «Своя игра», выберите режим «По словарю» и нужный вам словарь и нажмите «Поехали».
