# Архив спринтов

Здесь хранятся завершённые спринты для ретроспективного анализа.

## Формат имени файла

`sprint-YYYY-MM-DD-to-YYYY-MM-DD.md`

Пример: `sprint-2025-12-20-to-2026-01-03.md`

## Структура архива

Каждый спринт содержит:
- Цели спринта
- Выполненные задачи
- Метрики (план vs факт)
- Ретроспективу
- Уроки для будущих спринтов
